Here compare React.js and Vue.js functionality.
- 
Both applications was created by:

- React.js `create-react-app`
- Vue.js `vue-cli`

For the start an applications you should go to each folder and run npm scripts.

**React.js**:

 - `cd react_todo`
 - `npm start`
 
**Vue.js**:

 - `cd vue_todo`
 - `npm run serve`
